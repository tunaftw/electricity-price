# Visa elprisstatus

Visa status för nedladdad elprisdata.

## Instruktioner

Kör `python3 status.py` och presentera resultatet för användaren.

Förklara:
- RAW DATA: Rådata från API:et (blandad upplösning)
- QUARTERLY DATA: Processad data i 15-min upplösning (för analys)

Om quarterly-data saknas, föreslå att köra `python3 process.py`.
