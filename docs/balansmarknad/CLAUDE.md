# Instruktioner för Balansmarknads-kunskapsbanken

## Arbetssätt

När vi diskuterar balansmarknaden och jag tar fram **ny officiell information** som inte redan finns dokumenterad i kunskapsbanken, ska jag:

1. **Identifiera ny information** - Notera när sökt/hämtad information saknas i befintliga dokument
2. **Fråga dig** - "Ska vi lägga till denna information i kunskapsbanken?"
3. **Uppdatera rätt fil** - Lägg till i relevant befintlig fil eller skapa ny vid behov
4. **Uppdatera översikten** - Håll innehållsförteckningen nedan aktuell
